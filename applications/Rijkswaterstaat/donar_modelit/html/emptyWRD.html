Function emptyWRD

  Make default WRD (Waarde administratie) block.

CALL:

   WRD = emptyWRD

INPUT:

    No input required.

OUTPUT:

   WRD: Structure with fields:
      +----taxis (double)  
      +----lKeynr2 (double)
      +----Wrd (double)    
      +----nKwlcod (double)

EXAMPLE:

   WRD = emptyWRD
WRD = 
      taxis: []
    lKeynr2: []
        Wrd: []
    nKwlcod: []
