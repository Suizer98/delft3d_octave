Function emptyblok

  Make an empty Donar data block.

CALL:

   blok = emptyblok

INPUT:

   No input required.

OUTPUT:

   blok: Donar data block, with the following required partial data blocks:
         - W3H
         - RKS
         - WRD (must contain at least one row of data)
  
         Optional partial data blocks:
         - MUX
         - TYP
         - TPS

EXAMPLE:

   blok = emptyblok
blok = 
    W3H: [1x1 struct]
    MUX: []
    TYP: []
    RGH: []
    RKS: [1x1 struct]
    TPS: []
    WRD: [1x1 struct]
