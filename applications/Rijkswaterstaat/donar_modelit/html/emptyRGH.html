Function emptyRGH

  Make default RGH (Raaigeldigheid administratie) block.

CALL:

   RGH = emptyRGH(blok)

INPUT:

   blok: (Optional) structure with DIA block, 
         if specified create RGH blok based on known RGH blocks.

OUTPUT:

RGH:

    Structure with fields:
      +----lBegdat: begin datum
      +----lEnddat: einddatum (0: oneindig)
      +----Raaitype (sRaityp):
      |    H: hulpstrandhoofdraai
      |    S: strandhoofdraai
      |    N: normale raai
      +----Raaiklasse (sRaikls):
      |    L: landelijke raai
      |    V: vaklodingsraai
      |    P: ?

EXAMPLE:

   RGH = emptyRGH
RGH = 
    lBegdat: 10000101
    lEnddat: 0
    sRaityp: 'N'
    sRaikls: 'L'
