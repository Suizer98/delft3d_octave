Function emptyMUX

  Make default MUX (Multiplex administratie) block.

CALL:

   MUX = emptyMUX

INPUT:

    No input required.

OUTPUT:

   MUX:  Structure with fields:
         +----iKannum (double)
         +----lWnsnum (double)
         +----sParcod (char)  
         +----sParoms (char)  
         +----sCasnum (char)  
         +----sStaind (char)  
         +----nCpmcod (double)
         +----sCpmoms (char)  
         +----sDomein (char)  
         +----sEhdcod (char)  
         +----sOrgcod (char)  
         +----sOrgoms (char)  
         +----sHdhcod (char)  
         +----sHdhoms (char)  
         +----sSgkcod (char)

EXAMPLE:

   MUX = emptyMUX
MUX = 
    iKannum: 1
    lWnsnum: 0
    sParcod: ''
    sParoms: ''
    sCasnum: ''
    sStaind: ''
    nCpmcod: 0
    sCpmoms: ''
    sDomein: ''
    sEhdcod: ''
    sOrgcod: ''
    sOrgoms: ''
    sHdhcod: ''
    sHdhoms: ''
    sSgkcod: ''
