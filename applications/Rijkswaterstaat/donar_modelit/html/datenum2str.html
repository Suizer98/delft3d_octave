Function datenum2str

  Convert a Matlab datenum to a string with format: yyyymmddHHMM.

CALL:

   str = datenum2str(datum)

INPUT:

   datum: Matlab datenum represention of date.

OUTPUT:

   str: String with date of form: yyyymmddHHMM.

EXAMPLE:

   str = datenum2str(now)
str =
201112060345
