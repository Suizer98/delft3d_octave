Function duration

  Calculate duration of a timeunit in Matlab datenum units.

CALL:

   d = duration(timeunit)

INPUT:

   timeunit: String with possible values:
             - 'mnd' months
             - 'd'   days
             - 'min' minutes
             - 'uur' hours
             - 'cs'  centiseconds

OUTPUT:

   d: Duration of the given timeunit in Matlab datenum units.

EXAMPLE:

   d = duration('d')
   d = duration('uur')
   d = duration('min')
d =
     1
d =
    0.0417
d =
  6.9444e-004
